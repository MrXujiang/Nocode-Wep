## 特点

- Nocode/Doc 文档搭建引擎
- Nocode/Web web应用搭建引擎
- Nocode/PPT 演示文稿搭建引擎
- Nocode/AI 基于办公场景的AI能力基座

## 更新记录

- **文档支持配置弹幕**
- **支持文章全局评论**
- **支持图文音视频**
- **支持划线评论**
- **支持自定义画板**
- **支持流程图编辑器**
- **支持表单设计器**
- **支持关键词注释，批注**
- **支持文章数据统计**（访问量，PUV，字数和阅读时间分析）
- **支持文章自定义赞赏配置**
- **支持json导入导出**
- **支持一键复制内容块**
- **支持无限层级目录**
